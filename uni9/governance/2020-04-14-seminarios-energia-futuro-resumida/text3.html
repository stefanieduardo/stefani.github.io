Transformação Digital

  • Dados com Qualidade
  • Big Data Corporativo

Notes:

As tecnologias habilitadoras (ver Capítulo 3 - Governança Digital 4.0), entre elas Inteligência Artificial, Machine Learning e Deep Learning precisam de dados com qualidade. Se uma empresa ou corporação não tiver uma estratégia para garantir a qualidade dos dados, outras estratégias como por exemplo uma campanha de publicidade ou o lançamento de um novo produto serão afetadas. É importante notar que os dados sempre existiram. Eles Sempre existiram, mas em razão de vários fatores como cultura organizacional ou competições internas, estavam presentes em silos para cada área de uma corporação. Em verdade travava-se um combate para verificar quem mais tinha dados. O Big Data Corporativo é um modo de criar uma unicidade. Dados de toda a corporação presente em uma única fonte. Isso permite que áreas diversas tenham acesso ao mesmos dados e isso permite uma processo de inovação aderente à realidade.
